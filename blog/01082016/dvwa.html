Damn Vulnerable Web Application | Docker

Damn Vunerable Web Application é uma aplicação PHP/MySQL com vários bugs para ir treinando as habilidades que se está estudando, podendo ser usada para ensinar sobre as falhas na faculdade, cursos…

Percebi que algumas pessoas tinha dificuldade para fazer a instalação dessa aplicação, e que tinha uns videos demonstrado como fazer, mas, ainda assim algumas pessoas ficavam com dificuldades, com isso pensei em criar uma imagem no docker para facilitar.

Mas como no mundo de SL tudo acaba sendo muito mais rápido um cara já fez, então mostrarei apenas como instalar e começar a utilizar.
Primeiro vamos precisar do Docker e aqui está a Documentação

Para instalar recomendo usar o wget em distruibuições baseadas em Debian por que o pacote no repo oficial do Debian está desatualizado.

$ wget -qO- https://get.docker.com/ | sh

Depois desse processo doloroso, só baixar a imagem:

$ docker pull citizenstig/dvwa

Subindo a máquina

$ docker run -d -p 80:80 citizenstig/dvwa

Então o docker facilita a vida de qualquer um, sua aplicação está no ar em menos de 2 minutos :P

para obter um shell de dentro da máquina só digitar.

$ docker exec -it <CONTAINER ID > bash

o login para acessar via browser é admin e senha password.

Depois disso só brincar a vontade.

Rubocop

O que é?

Rubocop é uma analisador de código estático para quem programa em ruby, é uma maneira de saber se o código está limpo, se está conforme o estilo rubysta pede :P
Ele é baseado no Rusby Style Guide .

O que torna essa gem uma ferramenta muito interessante, já que você pode manter seu código limpo e também faz com que os contribuidores do seu projeto mande apenas códigos limpos. Desse jeito vai ser muito mais easy pra novos contribuidores entender seu projeto.

Como instalar

Pra instalar é bem simples bastar digitar: gem install rubocop, mas recomendo adcionar ao seu arquivo Gemfile pois assim quem for contribuir para seu projeto pode simplesmente rodar um 'bundle install' e vai ter ele.

Seu Gemfile deve ser uma linha mais ou menos assim:
gem install 'rubocop', require: false

Use e repita, e repita e repita...

Pra rodar o rubocop é bem simple é só digitar rubocop Não se assuste com o número de 'ofensas' que pode vir a ser bem grande.

Eu recomendo criar um arquivo .rubocop.yml na raiz do seu projeto, assim você pode adcionar parametros que se adeque melhor ao seu projeto.
Por exemplo: O tamanho máximo que sua linha deve ter, o tamanho máximo que seus métodos devem ter, pastas que não devem ou devem ser testadas.
